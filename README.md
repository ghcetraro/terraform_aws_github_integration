# terraform_aws_github_integration

This code is use to create an integration between aws and github to use roles to authenticate instead of username and password which is insecure  

This can be used in github actions to connect to aws and perform different types of actions  

## This repository creates:  

  .- github aws integration  
  .- roles and permissions  
  .- ecr
